Fungal infections (FI) are common and severe complications in hospitalized patients, with early identification of high-risk patients being critical for improving clinical outcomes. To address the limitations of existing models in accuracy and clinical applicability, this project develops and validates an interpretable machine learning model for predicting FI risk using multi-center data. Feature selection was performed using LASSO, Random Forest (RF), and SVM-RFE methods, followed by the development of five ML models including Logistic Regression (LR), RF, Decision Tree (DT), XGBoost, and LightGBM. The SHAP method was employed to enhance clinical interpretability, and an online prediction tool based on Streamlit (https://fiprediction.streamlit.app/) was developed to enable real-time risk assessment and personalized clinical recommendations. This repository contains the complete code for feature selection, model training, evaluation, and deployment.
